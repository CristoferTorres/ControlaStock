<p align="center">
ControlaStock es un sistema de inventarios para tiendas miscelaneas.
Permite registrar productos, actualizar existencias, eliminar articulos y consultar el stock disponible de manera rapida y sencilla.
Su objetivo es facilitar la gestion de inventarios para pequeñas tiendas, optimizando el control de productos y evitandno  faltantes o exceso de mercancia.
</p>
